# Abilities

Tasks and effects from the StateTreeToolsGameplayAbilitySystem plugin.

| Task | Description |
|------|-------------|
| [Try Activate Ability By Class](https://bsvino.github.io/statetreetools.github.io/tasks/try-activate-ability-by-class) | Activates a gameplay ability on an actor's ASC and waits until the ability ends. |
| [Send Gameplay Event To Actor](https://bsvino.github.io/statetreetools.github.io/tasks/send-gameplay-event-to-actor) | Sends a gameplay event to an actor's ASC on enter, exit, or both. |
| [Apply Gameplay Effect to Self](https://bsvino.github.io/statetreetools.github.io/tasks/apply-gameplay-effect-to-self) | Applies a gameplay effect to the bound actor's own ASC. |
| [Apply Gameplay Effect to Target](https://bsvino.github.io/statetreetools.github.io/tasks/apply-gameplay-effect-to-target) | Applies a gameplay effect from one actor's ASC to a separate target actor's ASC. |
| [Wait For Gameplay Tag Added](https://bsvino.github.io/statetreetools.github.io/tasks/wait-for-gameplay-tag-added) | Waits until a gameplay tag is added to an actor's ASC. |
| [Wait For Gameplay Tag Removed](https://bsvino.github.io/statetreetools.github.io/tasks/wait-for-gameplay-tag-removed) | Waits until a gameplay tag is removed from an actor's ASC. |
