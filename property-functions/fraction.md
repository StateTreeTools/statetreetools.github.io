# Fraction

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns the fractional (decimal) part of a float — the portion after the decimal point. For example, `3.75` produces `0.75`. Use this to extract a sub-unit portion of a value, such as isolating the progress within the current second of an elapsed time.

---

## Configuration

### Value
The float to extract the fractional part from.

### Result
The fractional portion of **Value**, always in the range `[0, 1)`.

## Unreal Engine Version Notes

Property functions are not available in UE 5.4. This node requires **UE 5.5 or later**.
