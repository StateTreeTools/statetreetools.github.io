# Acos (Radians)

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns the arccosine (inverse cosine) of a value, expressed in radians. The input must be in the range `[-1, 1]`; values outside this range produce `NaN`. Commonly used to recover an angle from a dot product of normalized vectors.

---

## Configuration

### Value
The cosine value to invert. Must be in `[-1, 1]`.

### Result
The arccosine of **Value** in radians, in the range `[0, π]`.

## Unreal Engine Version Notes

Property functions are not available in UE 5.4. This node requires **UE 5.5 or later**.
