# Tan (Radians)

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns the tangent of an angle given in radians. The result is unbounded and approaches infinity as the angle approaches ±π/2.

---

## Configuration

### Value
The angle in radians.

### Result
The tangent of **Value**.

## Unreal Engine Version Notes

Property functions are not available in UE 5.4. This node requires **UE 5.5 or later**.
