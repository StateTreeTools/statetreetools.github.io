# Max

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns the larger of two float values. Use this to ensure a value never drops below a floor — for example to guarantee a timer always starts with at least some minimum duration, or to pick the longer of two distances.

---

## Configuration

### Left
The first float value to compare.

### Right
The second float value to compare.

### Result
The larger of **Left** and **Right**.

## Unreal Engine Version Notes

Property functions are not available in UE 5.4. This node requires **UE 5.5 or later**.
