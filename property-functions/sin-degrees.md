# Sin (Degrees)

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns the sine of an angle given in degrees. Use this when your angle is already in degrees, avoiding the need for a separate [Degrees To Radians](/property-functions/degrees-to-radians) conversion.

---

## Configuration

### Value
The angle in degrees.

### Result
The sine of **Value**, in the range `[-1, 1]`.

## Unreal Engine Version Notes

Property functions are not available in UE 5.4. This node requires **UE 5.5 or later**.
