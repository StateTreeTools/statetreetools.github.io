# Cos (Radians)

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns the cosine of an angle given in radians. Use this with angular values already expressed in radians, such as those produced by [Degrees To Radians](/property-functions/degrees-to-radians).

---

## Configuration

### Value
The angle in radians.

### Result
The cosine of **Value**, in the range `[-1, 1]`.

## Unreal Engine Version Notes

Property functions are not available in UE 5.4. This node requires **UE 5.5 or later**.
