# Asin (Degrees)

**Plugin:** StateTreeToolsCore
**Category:** Math \| Float

Returns the arcsine (inverse sine) of a value, expressed in degrees. The input must be in the range `[-1, 1]`; values outside this range produce `NaN`.

---

## Configuration

### Value
The sine value to invert. Must be in `[-1, 1]`.

### Result
The arcsine of **Value** in degrees, in the range `[-90, 90]`.

## Unreal Engine Version Notes

Property functions are not available in UE 5.4. This node requires **UE 5.5 or later**.
