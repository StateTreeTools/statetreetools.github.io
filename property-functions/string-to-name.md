# String To Name

**Plugin:** StateTreeToolsCore
**Category:** Utilities \| String

Converts a plain string to an `FName`. Use this when you need to feed a string value into a property or function that expects an `FName` — for example a socket name, tag name, or component name.

---

## Configuration

### Value
The string to convert to a Name.

### Result
An `FName` with the same text as **Value**.

## Unreal Engine Version Notes

Property functions are not available in UE 5.4. This node requires **UE 5.5 or later**.
